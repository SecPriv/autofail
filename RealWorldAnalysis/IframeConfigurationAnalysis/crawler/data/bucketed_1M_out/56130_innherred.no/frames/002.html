Kjære Innherred-leser,

For at vi skal kunne gi deg en best mulig opplevelse når du besøker oss, og for å kunne forbedre våre tjenester og annonser, er vi avhengige av at du og andre deler data med oss. Dette bidrar til å finansiere journalistikken om ditt nærområde. Du finner mer informasjon om hva dette innebærer her.

Vi bruker cookies for blant annet å sikre kvaliteten på tjeneste våre. Når du godtar cookies, godtar du at vi og våre partnere, samt Schibsted Media AS og deres 32 partnere, kan bruke cookies, enhets-IDer og andre data som lagres og/eller vi får tilgang til på din enhet fra Innherred, til å samle opplysninger om deg, slik som informasjon om hvilke sider du har besøkt, for disse formålene:

Informasjonskapsler, enhetsidentifikatorer eller lignende nettidentifikatorer (f.eks. påloggingsbaserte identifikatorer, tilfeldig tildelte identifikatorer, nettverksbaserte identifikatorer) sammen med annen informasjon (f.eks. nettlesertype og informasjon, språk, skjermstørrelse, støttede teknologier osv.), kan lagres eller leses på enheten din for å gjenkjenne den hver gang den kobles til en app eller til et nettsted, for ett eller flere av formålene som presenteres her.

    Reklame kan tilpasses basert på profilen din. Aktiviteten din på denne tjenesten kan brukes til å bygge eller forbedre en profil om deg for personlig annonsering. Annonse- og innholdsytelse kan måles. Rapporter kan genereres basert på din og andres aktivitet. Aktiviteten din på denne tjenesten kan bidra til å utvikle og forbedre produkter og tjenester.

      Hvis du samtykker til det, kan vi samle inn og benytte opplysninger om din aktivitet på denne tjenesten, herunder sidevisninger og enhets-ID for å tilpasse og måle vårt innhold, utvikle våre tjenester, markedsføre våre produkter og tjenester, og annonsering. Dersom du logger inn på din profil, vil vi også sammenstille opplysningene om din aktivitet med din profilinformasjon for de samme formålene. Det betyr at hvis du ikke samtykker til behandling av atferdsdata for formålene ovenfor, kan vi likevel behandle profilinformasjonen for disse formålene. Du kan reservere deg mot at vi behandler profilinformasjonen din under Personverninnstillinger som du finner i brukermenyen.

        Schibsted behandler atferdsdata for vår annonseaktivitet, inkludert målgruppemålretting og segmentering. Hvis du samtykker til det, kan Schibsted Media AS samle inn og benytte opplysninger om din aktivitet på denne tjenesten for å bygge og forbedre en profil om deg, tilpasse reklame basert på profilen din, måle annonse- og innholdsytelse, generere rapporter basert på din og andres aktivitet, samt utvikle og forbedre produkter og tjenester. Dersom du logger inn på din profil, vil Schibsted Media AS også sammenstille opplysningene om din aktivitet med din profilinformasjon for de samme formålene. Det betyr at hvis du ikke samtykker til behandling av atferdsdata for formålene ovenfor, kan Schibsted Media AS likevel behandle profilinformasjonen for disse formålene. Du kan reservere deg mot at Schibsted Media AS behandler profilinformasjonen din under Personverninnstillinger som du finner i brukermenyen.

          Velg «Tilpass eller avvis» for å håndtere samtykkene dine. Her kan du også justere eller reservere deg mot at Schibsted Media AS og deres annonsepartnere behandler atferdsdata basert på berettiget interesse. Du kan når som helst endre disse innstillingene via lenken «Cookieinnstillinger» på tjenestens side.
