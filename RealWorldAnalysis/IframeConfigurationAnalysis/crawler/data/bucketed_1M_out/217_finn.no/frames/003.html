Cookieinnstillinger

Vi bruker cookies for å sikre kvaliteten på tjenestene våre. Når du godtar cookies, godtar du at Vend og våre 41 partnere kan bruke cookies, enhets-IDer og andre data som lagres og/eller vi får tilgang til på din enhet fra FINN, til å samle inn persondata, slik som informasjon om hvilke sider du har besøkt, for disse formålene:

Informasjonskapsler, enhetsidentifikatorer eller lignende nettidentifikatorer (f.eks. påloggingsbaserte identifikatorer, tilfeldig tildelte identifikatorer, nettverksbaserte identifikatorer) sammen med annen informasjon (f.eks. nettlesertype og informasjon, språk, skjermstørrelse, støttede teknologier osv.), kan lagres eller leses på enheten din for å gjenkjenne den hver gang den kobles til en app eller til et nettsted, for ett eller flere av formålene som presenteres her.

    Annonser kan personlig tilpasses basert på profilen din. Aktiviteten din på denne tjenesten kan brukes til å bygge eller forbedre en profil om deg for personlig tilpasset annonsering. Annonseytelse kan måles. Rapporter kan genereres basert på din og andres aktivitet. Aktiviteten din på denne tjenesten kan bidra til å utvikle og forbedre produkter og tjenester.

      Hvis du samtykker kan vi samle og bruke data om din aktivitet i denne tjenesten for egne formål, inkludert markedsføring av produktene våre gjennom plattformer som Google, Meta og Snap, annonsering til deg, personlig tilpassing av innhold, produktutvikling og analyser. Om du logger inn, slår vi sammen dataene med kontoinformasjonen din for de samme formålene.

        Velg “Tilpass eller avvis” for å håndtere samtykkene dine eller avvise behandlingen av data basert på berettiget interesse. Du kan når som helst endre innstillingene dine via lenken “Cookieinnstillinger” på tjenestens side.
