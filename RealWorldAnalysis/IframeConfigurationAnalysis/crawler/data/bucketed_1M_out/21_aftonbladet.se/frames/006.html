Cookieinställningar

Våra partners är förhindrade från att koppla ihop data som samlats in på denna app med data som samlats in från andra företags appar och webbplatser.

Genom att godkänna cookies bidrar du till att vi kan erbjuda relevanta tjänster och stärka annonsintäkterna som hjälper till att finansiera vår oberoende journalistik. Om du godkänner, så kommer Schibsted News Media AB och våra 39 partners att lagra och/eller få åtkomst till data på din enhet (t. ex. enhetsidentifierare) och behandla personuppgifter om dig (t. ex. vilka sidor du har besökt) från Aftonbladet för de syften som anges nedan.


Du kan avvisa personanpassad reklam för 49 kr/månad på reklamfinanserade Schibsted-tjänster. Du kommer fortfarande att se annonser, men de baseras inte på din aktivitet på våra webbplatser. Om du prenumererar på utvalda betalda redaktionella tjänster från Schibsted, kan du betala för att avvisa personanpassad reklam till det reducerade priset 39 kr/månad.

Du som har en annonsfri prenumeration ser förstås inga annonser på Aftonbladet, oavsett vilket val du gör här.

Syften

Vi och våra annons­partners ber om ditt samtycke för att behandla data för de syften som listas nedan.

Cookies, enhets- eller liknande onlineidentifierare (t.ex. inloggningsbaserade identifierare, slumpmässigt tilldelade identifierare, nätverksbaserade identifierare) tillsammans med annan information (t.ex. typ av webbläsare och information, språk, skärmstorlek, teknik som stöds osv.) kan lagras eller läsas på din enhet så att din enhet blir igenkänd varje gång den ansluter till en app eller till en webbplats, för ett eller flera av de ändamål som beskrivs här.

    Reklam kan personanpassas baserat på din profil. Din aktivitet i denna tjänst kan användas för att skapa eller förbättra en profil om dig för personanpassad reklam. Reklamens prestanda kan mätas. Rapporter kan genereras baserat på din och andras aktivitet. Din aktivitet i denna tjänst kan hjälpa till att utveckla och förbättra produkter och tjänster.
