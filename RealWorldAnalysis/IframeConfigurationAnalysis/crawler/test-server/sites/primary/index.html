Crawler iframe test scenarios

  • /plain.html — single same-origin iframe
  • /sandbox.html — iframes with various sandbox values
  • /allow.html — iframes with various allow (Permissions Policy) values
  • /srcdoc.html — iframe using srcdoc (inline HTML)
  • /nested.html — 3-level same-origin nesting
  • /dynamic.html — iframe injected by JavaScript after load
  • /cross-origin.html — iframe pointing to :3001
  • /cross-origin-nested.html — cross-origin frame that itself embeds another iframe
  • /data-url.html — iframe with a data: URL
  • /hidden.html — display:none iframe (CMP/TCF style)
  • /all.html — everything on one page
